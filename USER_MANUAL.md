## 🚀 Getting Started

AriTyper is a professional auto-typing software that extracts text from PDF or Word documents and types it into any application or browser window - even those that don't allow pasting.

### Key Features:
- ✅ **Multi-format Support**: PDF, DOCX, TXT files
- ✅ **Format Preservation**: Maintains text alignment (center, left, right, justified)
- ✅ **Window Selection**: Type into any application window
- ✅ **Adjustable Speed**: Control typing speed from ultra-fast to human-like
- ✅ **Device Locking**: Secure license system

## 📝 Basic Usage

### Step 1: Load Your Document
1. Click **"📂 Load File"** button
2. Select your document (PDF, DOCX, or TXT)
3. Review the extracted text in the preview area
4. Formatting and alignment are automatically preserved

### Step 2: Select Target Window
1. Click **"🪟 Select Window"** button
2. Browse through available application windows
3. Double-click to select your target window
4. The window name will be displayed

### Step 3: Configure Settings
1. **Typing Speed**: Adjust slider (1-100)
   - 1-10: Ultra-fast typing
   - 11-30: Fast typing
   - 31-60: Normal speed
   - 61-100: Slow, human-like typing

### Step 4: Start Typing
1. Click **"🚀 START TYPING"** button
2. You have 5 seconds to click into your target application
3. AriTyper will automatically type the content
4. Monitor progress in the progress bar

### Step 5: Stop if Needed
- Click **"⏹️ STOP"** to halt typing
- Move mouse to top-left corner for emergency stop

## 🔧 Troubleshooting

### Common Issues:

#### ❌ "License Not Activated"
**Solution**: 
- Check your transaction ID
- Ensure payment was completed
- Contact support if approval takes >30 minutes

#### ❌ "Window Not Found"
**Solution**:
- Refresh window list
- Ensure target application is open
- Select window before starting typing

#### ❌ "Typing Not Working"
**Solution**:
- Click into target field within 5 seconds
- Check if target window is still active
- Adjust typing speed if too fast

#### ❌ "File Not Supported"
**Solution**:
- Ensure file is PDF, DOCX, or TXT
- Check if file is corrupted
- Try converting to different format

#### ❌ "Device Locked"
**Solution**:
- License is device-specific
- Contact support for device transfer
- Purchase new license if needed

### Error Messages:
- **"Window selection failed"**: Restart application and try again
- **"Failed to load file"**: Check file format and permissions
- **"Typing error"**: Ensure target application is ready
